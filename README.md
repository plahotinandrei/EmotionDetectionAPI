# EmotionDetectionAPI

API для классификации эмоций в русских текстах с помощью BERT.

## Описание

Проект представляет собой api, который:
- принимает короткий текст на русском языке от пользователя;
- анализирует его с помощью дообученной модели `DeepPavlov/rubert-base-cased`;
- возвращает одну из шести эмоций: радость, грусть, удивление, страх, гнев или нейтрально с уверенностью.

## Функциональные возможности

- Отправка текска POST запросом.
- Определение эмоции.
- Отображение результата с указанием вероятности.

1. Проведите усатновку зависимостей: ```pip install -r requirements.txt```
2. Запустить через консоль с помощью команды: ```python main.py```
3. Ссылка на Swagger http://127.0.0.1:8080/docs
